JS Player — Juhi Sohal Player

A lightweight, mobile-first video player for local files & YouTube.

⬇ Download Android APK ▶ Try it in your browser

The APK is a debug build for direct install / testing. The Google Play release is produced separately by the signed release pipeline.

📲 Install the APK

  1. Tap Download Android APK above on your phone.
  2. Open the downloaded js-player.apk. Android will ask to allow installing from this source — enable Install unknown apps for your browser/file manager.
  3. Tap Install, then open JS Player.

Minimum Android 7.0 (API 24). If an older debug build is already installed, uninstall it first.

🍏 Install on iPhone / iPad

Apple requires every on-device app to be code-signed, so there's no sideloadable iOS download like the Android APK. Install the app as a PWA instead — it runs full-screen like a native app:

  1. Open the app in Safari on your iPhone/iPad.
  2. Tap the Share button (the square with an arrow).
  3. Choose Add to Home Screen → Add.

For a true App Store / TestFlight build, the signed publish-ios release pipeline is used (requires an Apple Developer account).

✨ Features

  • Local .mp3 songs (Music)
  • Local .mp4 videos (Video)
  • Album art & now-playing screen
  • Shuffle, repeat, prev / next
  • Seek, mini-player, auto-advance
  • No ads · no tracking · works offline

JS Player plays only the local files you choose. Nothing is uploaded, tracked, or sent anywhere — no ads, no accounts, no network needed.
